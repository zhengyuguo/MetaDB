How to search for public experiments

Search by keyword

Enter an keyword in the text box at top left corner of home page. Keep other drop down boxes blank to search by only keyword. Other search criteria e,g; gene, disease are AND conditioned with keyword.

Search by gene name or disease name or tissue type

Below keyword text box at home page, there are three drop down boxes for gene name, disease name and tissue type. Select your desired gene name, disease name or tissue type from the drop down boxes. These three search criteria are OR conditioned with each other and AND conditioned with keyword. You can opt out any search criteria by leaving it blank. For example, to search by only gene name, select the gene name from drop down box and leave other search criteria blank. To search by gene name or disease name, select gene and disease and leave keyword and tissue type blank. Leaving every search fields blank will show every entry of the database. After every search the system will remember your search criteria and will pre fill those criteria. To nullify the effect make it blank.


Filter your search

After successful search you can even filter it with typing any word in search box at right side of home page to show the entries containing that word. To sort the table click the caret icon available at every column heading.
